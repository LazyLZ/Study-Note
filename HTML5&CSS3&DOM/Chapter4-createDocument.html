创建文档--元数据元素

文档元素和元数据元素是用来创建文档和说明其内容的元素。元数据元素应该全部放在head标签内部，用于提供文档信息，外联CSS、脚本或 图标等。

  • title: 用于设置文档标题，必不可少。
  • base: 用于设置URL相对解析基准和打开页面中所有链接的方式。base需使用自闭和标签。
    <base href="http://www.baidu.com/" target="_blank"/>
  • meta: 用来说明文档，可用于指定名/值数据对、声明字符编码、改写http标头（可用于重载页面） 
    <meta name="author" content="lz"/>
<meta charset="UTF-8"/>
<meta http-equiv="refresh" content="5"/>
  • style: 用于定义内嵌样式表（link定义外部样式表。
    <style type="text/css">
    a{color: red; padding: 0.5em; display: inline-block;}
</style>
  • link: 用于在HTML文档和外部资源之间建立联系，用于引入图标，样式表等。 
    <link rel="stylesheet" type="text/css" href="styles.css"/>
<link rel="icon" type="image/x-icon" href="favicon.ico"/>

    注：type为文件的MIME类型和文件后缀名有关但不相同。
  • script: 在文档中嵌入脚本。src属性指定文件地址，defer/async指定加载模式，均为可选。

    例1：内部脚本

     <head>
    <script>
         function sayHi(){
             alert("Hi");
         }
          sayHi();
    </script>
</head>

    例2：外部脚本

    <head>
    <script src="test.js"></script>
</head>
  • noscript: 处理在浏览器关闭或不支持脚本的情况。
    <noscript>
    <h1>JavaScript is requaired</h1>
</script>
