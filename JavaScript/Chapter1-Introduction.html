第一章--JavaScript简介

  1. 完整的JavaScript实现应该由核心ECMAScript，DOM（Document Object Model），BOM（Browser Object Model）组成
  2. ECMAScript由ECMA-262定义，与web没有依赖关系，web只是ECMAScript实现的宿主环境之一，其余宿主环境还有Node.js，Adobe flash
  3. ECMA-262规定了JavaScript的语法、类型、语句、关键字、保留字、操作符和对象
  4. DOM是针对XML但经过扩展用于HTML的API（Application Programming Interface）DOM将整个页面映射为一个多层节点结构，HTML或XML的每个组成部分都是某种类型的节点。
  5. BOM可以控制浏览器显示的页面以外的部分，即处理浏览器的窗口和框架。BOM没有标准可以遵循，但仍朝着兼容性越来越高的方向发展。
