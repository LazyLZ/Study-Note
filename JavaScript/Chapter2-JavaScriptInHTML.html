第二章--在HTML中使用JavaScript

在HTML中插入JavaScript的方法，主要是使用<script>标签,<script>定义了六个属性:

  • async:异步脚本
  • defer:延迟脚本，仅外部脚本有效
  • charset:指定代码字符集，极少用
  • language:已废弃
  • src:可选，指定包含外部脚本的文件路径
  • type:脚本类型，默认JavaScript

例1：内部脚本

 <head>
    <script>
         function sayHi(){
             alert("Hi");
         }
          sayHi();
    </script>
</head>

例2：外部脚本

<head>
    <script src="test.js"></script>
</head>
